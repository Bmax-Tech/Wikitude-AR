***************************** JURASSIC PACK *****************************
------------------------------------------------------------------------------------------------------
    
	Thank you for purchasing this model/Dino Pack. 
    
	This asset contains 3 kind of folders, dinosaurs, eyes and world folder.
	
	"dinosaur name" folders:
	The FBX model with animations, textures, controllers, prefabs and a C# script allowing
	the dino to move in all direction, play animations and sounds.

	"eyes" folder :
	Shared eyes textures for dinos.

	"World" folder :
	Contain the demo scene, unity terrain, trees, grass, ground textures... and
	a script used to control the camera, display tools like fullscreen, fps, wireframe mode, model scale
	and other stuffs to play around all dinos.
